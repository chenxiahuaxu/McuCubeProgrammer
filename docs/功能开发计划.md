# 功能开发计划

> v0.0.1 已完成基本烧录 + SWO 调试输出。

---

## 已完成 ✅

- 探针扫描（ST-Link / DAPLink / J-Link 自动识别）
- 烧录流程（扇区擦除 → 烧录 → 验证 → 复位）
- SWO 实时串行调试日志
- 两级芯片选择（厂家 + 搜索输入）
- CMSIS-Pack 安装
- 配置持久化（探针 / 芯片 / 固件记忆）
- 烧录进度 + 扇区/地址详情日志
- 多平台（桌面 + Web）
- 深色 PCB 精密仪器主题
- 多语言支持（中/英自动检测 + 手动切换）
- 擦除模式选择（扇区擦除 / 全片擦除）

| 1 | 芯片内存 hex dump | 连接后读取 Flash 任意地址，以 `0x08000000: xxxxxxxx` 格式展示 |

| 5 | elf 解析 | 加载 elf 后列出全局变量符号表，自动提取目标芯片型号 |
